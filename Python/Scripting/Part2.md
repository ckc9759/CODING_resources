#### Lists

```py

reverse()
```

#### Dictionary

```py
key value pair
get() --> to get the value of a key in dictionary
keys() --> keys in dicitionary
values() --> values in dictionary
items() --> key, value pair
update() --> Similiar to extends() function of list

popitem() --> pops the last element
```
